스드메 패키지 vs 분리 계약 — 어떻게 다른가

스튜디오·드레스·메이크업을 묶음으로 할지 따로 할지. 한국 신혼부부가 가장 많이 고민하는 결정 중 하나예요. 가격·품질·시간 비교 정리.

스드메는 평균 380~500만원이 드는 항목이에요. 웨딩홀 제휴 패키지(번들)로 묶으면 편하지만 옵션 부풀리기로 비싸지기 쉽고, 따로 계약하면 가격은 줄어드는 대신 발품을 팔아야 해요. 둘의 차이를 정리했습니다.

한눈에 비교

📦 패키지 (번들)

  • 편함 — 한 번에 다 정해짐
  • 웨딩홀 제휴 → 할인
  • 의외의 옵션 부풀리기 (+50~150만원)
  • 품질 일관성 보장 X
  • 변경·취소 어려움
  • 총 비용 평균 450~550만원

🔍 분리 계약

  • 발품 필요 (3~5곳 방문)
  • 가격 비교로 50~150만원 절감
  • 각 분야 1위 업체 골라잡기
  • 본인 취향에 맞춤
  • 일정 조정 부담
  • 총 비용 평균 350~450만원

패키지의 함정 — "옵션 부풀리기"

웨딩홀 제휴 패키지에서 가장 많이 발생하는 추가 비용 패턴:

기본 패키지 금액추가되는 옵션실제 결제액
299만원드레스 등급업 50, 본식 메이크업 30, 원판 사진 80459만원
399만원드레스 추가 25만, 액자 60, 헤어팩 25, 친정엄마 메이크업 30539만원
499만원스튜디오 의상 추가 40, 한복 30, 부케 25, 본식 출장 메이크업 50644만원

광고된 가격 + 실제 결제액 사이 차이가 평균 100~200만원이에요. 이건 사기는 아니고, 기본 패키지에 포함 안 된 항목을 안내하는 정상 영업이지만 미리 모르면 당황스럽습니다.

"기본 패키지가 300만원이면 진짜 결제는 450만원 안팎이에요. 광고가만 보고 결정하지 말고, 실제 결제 시나리오까지 계약 전에 물어봐야 해요."

분리 계약 — 어떻게 진행하나

① 스튜디오 (촬영)

  • 예산: 150~250만원 (등급별)
  • 방법: 인스타·웨딩북에서 사진 스타일 보고 후보 3곳
  • 방문 → 견적 비교 → 계약
  • 의상 컨셉(노출·실내·야외)도 결정

② 드레스 (대여)

  • 예산: 100~200만원 (본식 + 촬영 = 2벌이 보통)
  • 방법: 강남 압구정·청담 드레스샵 2~3곳 시착
  • 가봉 일정 미리 잡기 (예식 1개월 전)
  • 피팅 비 별도 발생 가능

③ 메이크업

  • 예산: 50~100만원 (본식 + 촬영)
  • 방법: 메이크업샵 또는 출장 메이크업
  • 본식 출장 메이크업은 30만원 추가 흔함
  • 친정엄마·신랑 메이크업도 같이 견적

어떤 부부에게 어떤 방식이 맞나

패키지가 맞는 부부

  • 준비 시간이 부족한 부부 (D-3~6개월)
  • 완성된 품질 일관성이 더 중요한 부부
  • 웨딩홀 제휴 할인이 의미 있는 경우 (10% 이상)
  • 고민·결정에 스트레스받는 성향

분리 계약이 맞는 부부

  • D-6개월 이상 시간 있는 부부
  • 가성비 추구·자기 취향 명확
  • 인스타·웨딩북·블링 같은 플랫폼 잘 다루는 편
  • 발품 안 아까운 성향

실전 체크리스트

패키지 계약 시

  1. "기본 패키지에 포함되는 것" 리스트 받기
  2. "흔히 추가되는 옵션 + 평균 가격" 미리 물어보기
  3. 변경·취소 위약금 조항 확인
  4. 웨딩홀 제휴 할인 % 확인
  5. 총 예상 결제액 (기본 + 평균 옵션) 계산해보기

분리 계약 시

  1. 스튜디오 2~3곳 견적 받고 비교
  2. 드레스 시착 일정 미리 잡기
  3. 메이크업 출장비 추가 여부 확인
  4. 각 계약서 위약금·일정 변경 조건 체크
  5. 총 예산 vs 패키지 견적 비교
